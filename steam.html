User Selectable World Tendency
Place one of the following messages anywhere in the world

  • Requesting a challenger... = Pure Black Tendency
  • You'll need a Soul Level of 10 ahead. = Pure White Tendency
  • Write more messages! = Neutral Tendency
  • Farewell! = Use Global Tendency System
Notice the message will not be placed in the world, But the Custom Code did work. Quit game, and Load back in.
